# Test file Overview
We have two test files:

* The main test file `test.xml`:

  It's "the" test file we normally speak of. All automated
tests in this module are based on it.

* The "virgin" test file `virgin.kmy`:

  A practically empty 
  KMyMoney
  file, as generated by the official program,
  with only five accounts (the default top ones) 
  and one currency
  (the base currency "EUR"). Pretty much the bare minimum 
  of what you can have.

# Main Test file
 
## Technical Aspects
The file has been generated with 
KMyMoney V. 5.2.2.

When you change the `test.xml` file, please save it in *uncompressed* XML format 
(by convention, compressed 
KMyMoney
files have the extension 
"kmy").

## Testing Aspects
Please be careful when making changes on the file: All JUnit test cases heavily depend on it, and you might break things.

# Virgin Test file
 
## Technical Aspects
The file has been generated with
